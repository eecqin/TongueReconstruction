The following are the data files used to demonstrate MOCHAtools (fsew0_*.*
from utterance '001' of speaker 'fsew0', extracted from the MOCHA-TIMIT
database with file format conversion):

- fsew0_001.wav: acoustic wave in RIFF format.
- fsew0_001.ema: EMA in ASCII format.
- fsew0_001.lab: phonetic label in ASCII format.
- fsew0_001.lar: laryngograph in RIFF format (not used in MOCHAdemo).

Additional files:
- fsew0_001.mov was created with MOCHAplot (see MOCHAdemo).
- fsew0_001.png was created with MOCHAscat (see MOCHAdemo).
